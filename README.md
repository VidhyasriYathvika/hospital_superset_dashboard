# Hospital Superset Dashboard

An interactive analytics dashboard for monitoring hospital operations, built using **Apache Superset**, **Python**, and **PostgreSQL**. This dashboard helps hospital administrators track **resource utilization**, **patient outcomes**, and optimize decision-making across departments.

## Features

- **Patient Admissions:** Monitor daily/weekly/monthly patient admissions.
- **Bed Occupancy:** Track bed usage and availability in real-time.
- **Doctor Workloads:** Analyze doctor appointments, surgeries, and workloads.
- **Procedure Volumes:** Visualize surgeries, lab tests, and other procedures.
- **Emergency Cases:** Identify trends in emergency visits and patient inflow.
- **Financial Overview:** Track billing, revenue, and operational costs.
- **Discharge Turnaround:** Measure patient discharge times for efficiency.
- **Interactive Visualizations:** Drill-down charts, filters, and time-series analytics.

## Tech Stack

- **Dashboard Tool:** [Apache Superset](https://superset.apache.org/)
- **Backend / ETL:** Python (pandas, SQLAlchemy)
- **Database:** PostgreSQL
- **Version Control:** Git & GitHub
